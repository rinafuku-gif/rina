## コンセプト

SATOYAMA AI BASEは「AI相談所」ではなく**「AI活用スクール」**。

- レベル制で入学し、卒業がある
- 目的は個人のスキルアップだけでなく、**地域全体のリテラシー底上げ**
- ずっと居座ってもらうのではなく、卒業して自走できる人を増やす
- 卒業した人が周囲にも広げる → 地域のDXが進む

## レベルシステム

既存の `ai-level-system.md` をベースに、サービス全体の軸として据える。

| フェーズ | レベル | テーマ |
|---|---|---|
| 体験フェーズ | Lv.1〜3 | 「何ができるか知る」 |
| スキルフェーズ | Lv.4〜5 | 「うまく使うコツを覚える」 |
| 応用フェーズ | Lv.6〜7 | 「使い道を広げる」 |
| システムフェーズ | Lv.8〜10 | 「仕組みにする」→ 卒業 |

## サイト構造（リニューアル案）

### 動線の基本設計

```
訪問者
  ↓
トップページ（レベル制の概要 + CTA）
  ↓
レベル診断（簡易5問くらい → 今のあなたはLv.○）
  ↓
「次のレベルに進むには？」→ プラン紹介
  ↓
メンバーシップ加入（Stripe）
  ↓
レベル別コンテンツ（学習教材）
  ↓
ワークショップ参加（体験会 → 定期開催）
  ↓
卒業（Lv.8〜10到達 → 自走可能）
```

### 既存ページの扱い

| 現ページ | リニューアル後 |
|---|---|
| ホーム | レベル制の紹介を軸にリデザイン |
| メンバーシップ | 3プランをセルフ/サポート/伴走に変更 |
| 思考のエンジニアリング | 無料ツールとして残す（集客用） |
| コンテンツハブ | レベルタグを追加、メンバー限定コンテンツ |
| イベント | ワークショップ = 体験会としての位置づけを明確化 |
| レベル診断 | **新規作成**（/diagnostic を転用 or 新ページ） |

## SNS発信戦略

### Instagram（satoyama_ai_base）

- **メイン投稿**: AI活用Tips・ハウツー（無料公開）
- **ストーリー**: ワークショップ告知、日常のAI活用風景
- **リール**: Before/After系の短い動画（AI使用前→使用後）

### 投稿のコンテンツソース

Ryoが日々の業務でAIを使って構築してきたものを、ターゲット層向けに分解してコンテンツ化。

既存の実績から生成可能なコンテンツ例:
- LINE公式アカウントでAI自動応答する方法
- レシート撮影で経費が自動記帳される仕組み
- 毎朝AIが今日のタスクをLINEで教えてくれる仕組み
- カレンダーの空きから日程調整テキストを自動生成
- 音声を送るだけで文字起こし＋要約
- AIへの「指示文のコツ」で業務メールが劇的に変わる話
- Googleスプレッドシートで顧客管理を始める方法
- Googleフォームでアンケートを作って自動集計する方法

### 発信ルール

- 全投稿はAIを活用して作成（サービスコンセプトの体現）
- Ryoが最終チェック → 修正もAIに指示
- 無料で公開する範囲と有料（スクール限定）の線引きを明確に
