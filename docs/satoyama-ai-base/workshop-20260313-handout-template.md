# 配布資料3: プロンプトテンプレート — 「明日使う指示文」を作ろう

## 基本テンプレート

```
あなたは ______________________________ です。
         （自分の事業の役割）

______________________________ に対して、
（誰に / どんな場面で）

______________________________ を
（何をしてほしい）

______________________________ で書いてください。
（どんなトーン・条件）
```

---

## 記入例

### 例1: カフェのSNS投稿
```
あなたは 山梨の小さなカフェの店主 です。
Instagram のフォロワー に対して、
今日の営業の様子を伝える投稿文 を
ほっこり温かい雰囲気で、ハッシュタグ5個つきで 書いてください。
```

### 例2: 教室の問い合わせ返信
```
あなたは ヨガ教室のインストラクター です。
体験レッスンに興味があるお客様 に対して、
体験レッスンの案内と予約方法を伝える返信 を
親しみやすく、3〜5行で 書いてください。
```

### 例3: 事業の壁打ち
```
あなたは 小規模事業のコンサルタント です。
一人で雑貨屋を経営している私 に対して、
売上が落ちている原因と改善案 を
具体的に、3つに絞って 提案してください。
```

## もっと良くするコツ

| やること | 効果 |
|---|---|
| 「3行で」「5案で」と数を指定する | 長すぎ・短すぎを防げる |
| 「〇〇な雰囲気で」とトーンを指定 | 自分らしい文章になる |
| 「ダメな例：〇〇」と注意点を伝える | 避けたい表現を防げる |
| 結果がイマイチなら「もう少し〇〇にして」と追加指示 | 会話しながら良くなる |
