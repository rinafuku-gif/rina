# 経費 勘定科目ルール

## 勘定科目一覧と判定ルール

### 仕入高
- コーヒー生豆、焙煎関連資材
- 民泊アメニティ・消耗備品の仕入れ

### 地代家賃
- えんがわ古民家賃料（年間50万円）
- コワーキング利用料
- レンタルスペース利用料

### 水道光熱費
- 電気、ガス、水道
- えんがわ・自宅の光熱費

### 旅費交通費
- ガソリン代、高速道路料金、ETC
- 電車・バス運賃
- 駐車場代
- 出張時の宿泊費

### 通信費
- 携帯電話料金
- インターネット回線
- サーバー・ドメイン費用（Vercel等）
- SaaS月額料金（Claude Max、その他ツール）

### 広告宣伝費
- SNS広告
- Airbnbプロモーション費用
- チラシ・名刺印刷
- Webサイト制作関連

### 接待交際費
- 取引先との飲食
- お土産・手土産
- 謝礼・お祝い

### 消耗品費
- 事務用品（文房具、プリンター用品）
- 10万円未満の備品・工具
- 清掃用品
- パソコン周辺機器（10万円未満）

### 新聞図書費
- 書籍、電子書籍
- 有料ニュース・情報サービス
- オンライン講座・セミナー

### 支払手数料
- Airbnb手数料
- Stripe決済手数料
- 銀行振込手数料
- 税理士・行政書士等の報酬

### 損害保険料
- 火災保険、地震保険
- 民泊向け賠償責任保険
- 自動車保険（事業按分）

### 修繕費
- 物件の修繕・メンテナンス
- 設備修理

### 車両費
- 車検・整備費用
- 自動車税
- 車両関連消耗品（タイヤ等）

### 外注工賃
- 業務委託費（デザイン、開発等）
- クリーニング外注

### 租税公課
- 個人事業税
- 印紙代
- 固定資産税（事業按分）

### 諸会費
- 商工会・業界団体の会費
- ADDress会費

### 福利厚生費
- 健康診断
- 事業に関連するレクリエーション

### 雑費
- 上記に該当しないもの

## 店名→勘定科目の自動判定ヒント

| 店名パターン | 勘定科目 |
|---|---|
| ガソリンスタンド、ENEOS、出光、コスモ | 旅費交通費（車両費でも可） |
| コンビニ、セブン、ファミマ、ローソン | 消耗品費 or 雑費（内容による） |
| スーパー、イオン、ヤオコー | 仕入高（民泊食材）or 雑費 |
| ホームセンター、コメリ、カインズ | 消耗品費 or 修繕費 |
| Amazon、楽天 | 内容による（消耗品費/新聞図書費/通信費） |
| 飲食店 | 接待交際費（事業関連の場合） |
| 東京電力、東京ガス | 水道光熱費 |
| NTT、ソフトバンク、au | 通信費 |
| 書店、紀伊國屋 | 新聞図書費 |

## マネーフォワード仕訳帳インポート形式

CSV列: 取引日,借方勘定科目,借方補助科目,借方税区分,借方金額,貸方勘定科目,貸方補助科目,貸方税区分,貸方金額,摘要,仕訳メモ,タグ

### 税区分
- 課税仕入10%（標準税率の経費）
- 課税仕入8%（軽減税率：食品等）
- 非課税仕入（保険料等）
- 対象外（租税公課等）

### 貸方科目
- 現金: 現金払い
- 事業主借: 個人のお金から立替
- 普通預金: 口座引落
- クレジットカード: 未払金
