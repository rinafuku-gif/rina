### 税区分
- 課税仕入10%（標準税率の経費）

### Ryo の決済方法（これ以外はほぼ使わない）

| 決済方法 | レシート上の表記パターン | 貸方勘定科目 |
|---|---|---|
| JCBデビット | JCB、デビット、J/デビット | 普通預金 |
| Mastercardデビット | Mastercard、マスターカード、MC、デビット | 普通預金 |
| PayPay | PayPay、ペイペイ、QR決済、バーコード決済 | 事業主借 |
| 現金 | 現金、お預り、おつり | 現金 |

### 支払方法の判定ルール
- レシートに「クレジット」「CREDIT」「デビット」「DEBIT」「JCB」「Mastercard」「MC」の記載がある → カード払い
- デビットカード（JCB/Mastercard）は即時引落なので貸方は「普通預金」（未払金ではない）
- 「PayPay」「ペイペイ」「QRコード」の記載がある → PayPay → 事業主借
- 「お預り」「おつり」の記載がある → 現金
- カード番号の末尾4桁が印字されている → カード払い
- 上記いずれにも該当しない場合 → レシートの内容から最も可能性の高いものを推定する

### 貸方科目まとめ
- 現金払い → 現金
- JCBデビット / Mastercardデビット → 普通預金
- PayPay → 事業主借
- クレジットカード（デビット以外） → 未払金
- 口座引落 → 普通預金
