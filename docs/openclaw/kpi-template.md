## 更新ルール

1. **更新者**: rina側のClaude Codeセッションで Ryo と一緒に更新
2. **更新頻度**: 週1回（週次レビュー時）+ 大きな変化があれば都度
3. **更新時のチェック**: 「この情報が第三者に見られたら困るか？」を毎回確認
4. **金額の書き方**: 絶対値NG。比率・達成率・前月比で表現する
5. **人名の書き方**: 役割で書く（「オーナー」「共同PM」「クライアント」）
6. **具体的すぎる取引先**: 伏せる（「ロケ検索サイト経由」はOK、「●●プロダクション」はNG）
