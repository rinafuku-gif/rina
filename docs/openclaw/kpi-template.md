## 更新ルール

1. **誰が更新するか**: rina側のClaude Codeセッションで Ryo と一緒に更新
2. **更新頻度**: 最低週1回（週次レビュー時）、大きな変化があれば都度
3. **絶対にやらないこと**: 具体的な金額（●万円、●円）を記載すること
4. **OKな表現**:
   - 「目標の62%」「前月比+15%」「事業A:事業B = 52:48」
   - 「稼働率68%」「予約3件」「進捗75%」
5. **NGな表現**:
   - 「月収20万円」「売上10.7万円」「報酬月額10万円」
