# iOSショートカット設定ガイド: 「しらたまに伝えて」

## 概要
「Hey Siri、しらたまに伝えて」と話しかけると、音声をテキスト化して
Discord経由でしらたまに伝わります。しらたまがボイスチャンネルにいれば
声で返事します。

## 設定手順

### 1. ショートカットアプリを開く
- iPhoneの「ショートカット」アプリを開く
- 右上の「＋」で新規作成

### 2. ショートカット名を設定
- 上部のタイトルをタップして **「しらたまに伝えて」** に変更
- これがSiriの呼び出しフレーズになる

### 3. アクションを追加

以下の順番でアクションを追加してください:

#### アクション 1: テキストを音声入力
- 「アクションを追加」→ 検索: **「テキストを音声入力」**
- 言語: **日本語**
- 聞き取り停止: **停止後**（話し終わったら自動で止まる）

#### アクション 2: URLの内容を取得
- 「アクションを追加」→ 検索: **「URLの内容を取得」**
- 設定:
  - URL: `https://api.tonari2tomaru.com/api/siri-input`
  - 方法: **POST**
  - 本文を要求: **JSON**
  - JSON の中身:
    - `text` → 「音声入力の結果」（前のアクションの出力を選択）
    - `token` → `（APIトークンをここに入力 — Ryoに直接伝えます）`

#### アクション 3: 結果を表示
- 「アクションを追加」→ 検索: **「結果を表示」**
- 表示内容: **「しらたまに伝えました！」**

### 4. 完成！

## 使い方

### 方法 1: Siri
「Hey Siri、しらたまに伝えて」

### 方法 2: ショートカットアプリから直接
ショートカットアプリで「しらたまに伝えて」をタップ

### 方法 3: ホーム画面に追加
ショートカットの「…」→ 共有ボタン →「ホーム画面に追加」

## 動作の流れ
1. Siriが音声を聞き取り → テキスト変換
2. テキストをMac miniのAPIに送信
3. APIがDiscord #generalに投稿
4. しらたまBotが検知してClaude AIで返答を生成
5. 返答をDiscordに投稿 + ボイスチャンネルで音声再生

## トラブルシューティング

### 「しらたまに伝えて」でSiriが反応しない
- ショートカット名が正確に「しらたまに伝えて」になっているか確認
- 「Hey Siri、しらたまに伝えてのショートカットを実行」と言ってみる

### 送信エラーが出る
- Wi-Fiまたはモバイルデータに接続されているか確認
- Mac miniが起動しているか確認
- APIトークンが正しいか確認

### 返答が来ない
- Discord #generalを確認（テキストでは返答が来ているはず）
- しらたまルームにいない場合、音声返答は再生されない（テキストのみ）
