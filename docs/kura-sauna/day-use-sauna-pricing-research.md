# 日帰り蔵サウナ 料金設定リサーチ（2026年3月）

## 調査目的

## 蔵サウナ＆ステイへの示唆

### ポジショニング案

蔵サウナ＆ステイは以下の強みがあり、中〜上位の価格設定が可能:
- **蔵のリノベーション**という希少性（全国でも数施設のみ）
- ADDress上野原A邸の敷地内という**里山ロケーション**
- 都心から約1.5時間のアクセス

### 料金設定案（日帰り貸切）

| プラン | 料金案 | 根拠 |
|---|---|---|
| 基本プラン（2時間・2名まで） | 8,000〜10,000円/組 | KURA:SAUNA土日祝相当、蔵の希少性を加味 |
| 追加1名 | +3,000〜4,000円/人 | KIKI・久米屋の追加料金水準 |
| ロングプラン（3時間・2名まで） | 12,000〜15,000円/組 | 2時間×1.5倍程度 |
| 平日割引 | 基本料金の20〜30%OFF | 平日/土日の価格差は業界慣行 |

### 検討ポイント

1. **定員上限**: 蔵のサイズに応じて最大4〜6名が現実的
2. **回転数**: 1日3〜4枠（準備・清掃含む）が運営上の目安
3. **付加価値オプション**: ドリンクセット、ロウリュアロマ、食事付きプラン等で客単価UP
4. **宿泊者特典**: 宿泊プランに日帰りサウナ無料 or 割引を組み込むとセット販売の動機に
5. **シーズン料金**: 冬季（サウナ需要高）は強気、夏季は水風呂・川の活用で差別化

### 収益シミュレーション（月間）

仮定: 基本料金 10,000円/組、1日平均2組、月20日稼働
- 月間売上: 10,000円 x 2組 x 20日 = **400,000円/月**
- 追加人数・オプション込みで **450,000〜500,000円/月** も視野

※ 宿泊事業との兼ね合いで枠数は調整が必要
