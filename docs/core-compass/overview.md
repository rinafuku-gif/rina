# CoreCompass / fate-decoder

## 基本情報

- 概要: 複数の占術体系を統合し、思考整理と行動指針を提供するWebアプリ
- Web: Vercel でホスティング（リポジトリ: fate-decoder）
- ステータス: 開発段階（未収益）
- 優先度: 中長期のストック収入として位置づけ

## 占術体系

- 算命学
- マヤ暦
- 数秘術
- 西洋占星術
- 宿曜

## コンセプト

「thinking engineering」— 複雑な思考パターンを解きほぐし、実行可能なロードマップを作成

## デザイン

「天体×和」の美学（星空アニメーション、金のアクセント）

## 収益モデル

- フリーミアム（基本無料 + 有料機能）
- PWAでの提供を検討中

## 派生アプリ構想

- イベントや飲み会で盛り上がる相性診断アプリ
- 本質的なキャリア診断アプリ
- いずれも占術計算ロジックを共通基盤として活用
